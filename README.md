# WebAnwProjekt
to start the web Project type in the following commands.

if you dont have docker installed, you have to install it first, else the Applications wont run on your system.  
-> important!! docker has to be active. 
```bash
git clone https://github.com/Mehmooo/WebAnwProjekt
cd WebAnwProjekt
docker compose build
docker compose up
```
